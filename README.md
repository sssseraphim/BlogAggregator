# RSS Blog Aggregator

Универсальный агрегатор RSS-лент для отслеживания ваших любимых блогов в одном месте.

##  Возможности

- **Подписка на RSS-ленты** - добавляйте любые RSS/Atom фиды
- **Автоматическое обновление** - новые статьи появляются автоматически  
- **Управление подписками** - подписывайтесь и отписывайтесь от блогов
- **Персонализация** - каждый пользователь видит только свои подписки
- **Высокая производительность** - написано на Go с оптимизированными запросамиe

##  Быстрый старт

### Предварительные требования

- Go 1.21+
- PostgreSQL 12+
- Git

### Установка

```bash
## Клонируйте репозиторий
git clone https://github.com/yourusername/BlogAggregator.git
```

### Настройка

Необходимо создать файл .gatorconfig.json в домашней дрикетории c

```json
{"db_url":"postgres://name:@localhost:5432/gator?sslmode=disable","current_user_name":"name"}
```

### Комманды

```bash
## Регистарция пользователя
./gator register {name}
## Логин пользователя
./gator login {name}
## Добавить фид
./gator addfeed {name} {url}
## Подписаться на фид
./gator follow {url}
## Собрать блог посты
./gator agg
## Показать num новых посты на которые подписан пользователь
./gator browse [num]
